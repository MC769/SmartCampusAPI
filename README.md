# Smart Campus REST API

## Overview
A JAX-RS RESTful API for managing university campus rooms and IoT sensors as part of the "Smart Campus" initiative.

## Technology Stack
- Java 17
- JAX-RS (Jersey 3.1.7)
- Grizzly HTTP Server (embedded)
- Maven
- In-memory ConcurrentHashMap (no database)

## Build & Run Instructions

### Prerequisites
```bash
java -version    # Java 17+
mvn -version     # Maven 3.6+
